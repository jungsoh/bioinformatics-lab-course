Setup

In your Linux home directory, make a new directory called lab02.

NCBI Entrez query
  1. NCBI Entrez is a portal to search NCBI databases. Starting from the portal, your task is to construct a query to retrieve all proteins that satisfy both of the following conditions:
    • Have a protein name “alcohol dehydrogenase 1”.
    • Come from either Rodents or Birds, but excluding the Mouse (Mus musculus) and all Rat species.
  2. On the Entrez main page, select an appropriate database to search against.
  3. On the selected database page, click “Advanced” to start building the query.
  4. Under “Builder”, specify your search conditions one by one. First pick an appropriate “Field” for the search, then enter the desired query term. Watch how the query string above Builder changes as you do. If the query was done correctly, about 200 proteins would be retrieved. Some tips:
    • You have four search conditions to enter.
    • You need to select one of AND, OR, and NOT when adding a search condition.
    • Use “OR” to indicate alternatives within a query term.
    • Organism names can be entered as common names, but using formal taxonomy terms will produce more accurate results. Use NCBI Taxonomy to find Latin taxonomy terms for common names like birds.
  5. How many entries are retrieved from the search? What is the most frequent protein length among the top 20 retrieved sequences?

BLAST word size
  1. Go back to the main nucleotide BLAST page. Click “Algorithm parameters” to show the BLAST parameters. Read the help information on “Word size”.
  2. Try increasing the word size from the default and run the BLAST again. Do you observe any effects on the BLAST result? What if you decrease the word size from the default?
  3. What are the general effects of the word size parameter in BLAST?
Local BLAST
  1. Save “Mystery_cDNA_sequence” into a text file called mystery.fasta in your lab02 directory. Then go to lab02.
  2. Search protein translations of this sequence against Swiss-Prot proteins, using local BLAST programs and the Swiss-Prot database. In the lab02 directory, run the blastx (translated nucleotide to protein BLAST) program with this command: 

    blastx -query mystery.fasta -db swissprot > blastx.txt

  3. When the command finishes, the BLAST results will be in the blastx.txt file. Use the command less blastx.txt to view its contents (Space for next screen, b to go backward, q to quit), and answer these questions:
    • What is the UniProt ID of the top hit?
    • Search for the ID in UniProt. Which organism is this top-hit protein from?
    • Look in “Cross-references” to find the NCBI RefSeq ID (or IDs) of this protein. Was this protein also reported as one of the top few hits in the previous NCBI Web BLAST?
    • In conclusion, is this local BLAST result similar to the NCBI Web BLAST result?
Simple query of UniProt
  1. Go to UniProt. What is the quickest way to search for proteins with the word “kinase” somewhere in its record from UniProtKB? How many entries are returned?
  2. Using the filters available under “Filter by”, filter the search result to list only those entries from human and with the term “kinase” in the “protein name”. How many entries do you have now?
  3. UniProtKB consists of both manually reviewed (Swiss-Prot) and unreviewed (TrEMBL) entries. How many reviewed entries were returned after the filtering?
Advanced query of UniProt
  1. In UniProt, click “Advanced” (and consult Help) to enter multiple search terms. Retrieve a set of about 100 proteins related to tumor, which satisfy the following conditions:
    • In Swiss-Prot
    • From a human genome
    • Has “tumor” somewhere in the text
    • Between 300 and 400 AA long
    • Involved in a disease
  2. UniRef90 is a database built by clustering sequences with at least 90% identity. From your query results, view UniRef90 clusters to which the retrieved sequences belong, and answer these questions:
    • How many UniRef90 clusters are listed?
    • What is the ID of the largest cluster? What is its size?
    • How many members does the smallest cluster have? How many organisms are represented in it?
